# 体育赛事实时赔率展示面板 — 产品需求文档（PRD）

## 1. 产品概述

体育赛事实时赔率展示面板是一款面向赛事数据爱好者与竞彩分析人群的高密度实时数据看板，提供今日/明日多联赛赛事对阵、全维度赔率（胜平负、亚盘让分、大小球、波胆、总进球）、投注资金分布与异常波动监测能力。
- **核心目的**：解决用户在多源数据下难以一站式追踪实时赔率变化与市场热度的问题，通过高频刷新与视觉反馈让用户秒级感知市场动向。
- **目标用户**：竞彩数据分析师、赛事投注参考者、体育数据爱好者。
- **市场价值**：以"指挥中心"级的信息密度与实时性，提供专业、可信、可决策的赛事赔率分析体验。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 访客 | 无需注册 | 浏览全部赛事、赔率、分布与异常预警 |

### 2.2 功能模块

1. **赔率看板主页（唯一核心页面）**：顶部状态栏（连接状态/时钟）、筛选与日期切换、赛事虚拟滚动列表、赛事卡片（对阵+赔率+分布+标签）、异常预警浮层。

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 赔率看板主页 | 顶部状态栏 | 实时时钟、WebSocket连接状态指示灯、在线赛事计数、异常波动总数 |
| 赔率看板主页 | 筛选与日期 | 今日/明日切换、联赛与运动类型筛选、搜索框、热门/异常快捷过滤 |
| 赔率看板主页 | 赛事列表 | 虚拟滚动渲染大量赛事，热门赛事置顶，按日期分组 |
| 赛事卡片 | 对阵信息 | 主/客队名称与logo、开赛时间（精确到分钟）、实时比分（进行中）、联赛标识 |
| 赛事卡片 | 赔率模块 | 胜平负（欧赔+亚盘让分）、大小球、让球盘、波胆、总进球数；实时刷新 |
| 赛事卡片 | 投注分布 | 主胜/平局/客胜资金占比横向进度条，悬停查看百分比 |
| 赛事卡片 | 标识系统 | "火热"热门标签、"异常波动"红色警示标记、进行中状态标签 |
| 赛事卡片 | 视觉反馈 | 赔率下降绿色闪烁、赔率上升红色闪烁、异常红色持续警示 |

## 3. 核心流程

用户进入看板 → WebSocket建立连接并加载初始赛事数据 → 按今日/明日与联赛筛选 → 浏览虚拟滚动赛事列表 → 热门赛事置顶展示 → 查看赔率与投注分布 → 赔率实时刷新触发颜色闪烁 → 异常波动触发警示标记 → 断线自动重连恢复数据流。

## 4. 用户界面设计

### 4.1 设计风格

- **整体风格**：体育数据指挥中心（Command Center），暗色终端质感，高密度信息，赛博与体育场夜场氛围融合。
- **主色调**：深空藏青黑 `#070A12` / 炭灰 `#0F1420`；强调色采用琥珀金 `#F5B614`（赛事主调）与电光青 `#22D3EE`（数据高亮）。
- **反馈色**：赔率下降（热度↑）用霓虹绿 `#00E676`；赔率上升（冷门↑）用猩红 `#FF2D55`；异常波动用深红 `#FF1744` 持续脉冲。
- **按钮样式**：极简胶囊/方形切角，激活态琥珀金描边发光，悬停微抬升。
- **字体**：标题与队名用 `Oswald`（运动感压缩体）；正文用 `Manrope`；赔率与比分数字用 `JetBrains Mono`（等宽数据终端感）。
- **布局**：卡片式栅格 + 虚拟滚动单列纵向列表，顶部固定状态栏，左侧筛选抽屉（移动端折叠）。
- **图标**：线性运动图标 + 状态圆点指示灯（连接/异常/进行中）。

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|----------|
| 赔率看板主页 | 顶部状态栏 | 暗色玻璃质感栏、闪烁连接圆点、等宽时钟、计数芯片 |
| 赔率看板主页 | 筛选区 | 日期分段控件、联赛多选芯片、搜索输入、热门/异常开关 |
| 赔率看板主页 | 赛事列表 | 虚拟滚动容器、日期分组标题、置顶热门卡片 |
| 赛事卡片 | 对阵头 | 队徽圆形容器、压缩体队名、时间/比分等宽数字、状态标签 |
| 赛事卡片 | 赔率网格 | 多列等宽赔率单元，每个单元含标签+欧赔+亚盘+闪动背景 |
| 赛事卡片 | 分布条 | 三段水平进度条（主胜/平/客胜），悬停Tooltip百分比 |
| 赛事卡片 | 标签层 | 琥珀金"火热"徽章、红色脉冲"异常波动"徽章 |

### 4.3 响应式

- **桌面优先**：宽屏下赛事列表支持多列并排卡片或宽幅卡片，赔率横向铺开。
- **移动自适应**：单列卡片，赔率模块纵向折叠，筛选抽屉从底部滑出。
- **触控优化**：分布条悬停Tooltip在移动端改为长按显示，点击区域≥44px。

### 4.4 可访问性

- 颜色对比度满足 WCAG AA（正文≥4.5:1），状态不仅依赖颜色，同时辅以图标与文字标签。
- 全量 ARIA 属性：`aria-live="polite"` 用于实时赔率播报，`aria-busy` 加载态，按钮与进度条语义化。
- 支持键盘导航：Tab 可聚焦赛事卡片与赔率单元，方向键切换，Enter 展开详情。
